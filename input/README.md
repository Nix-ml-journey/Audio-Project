# Input Folder

This folder is where you should place your audio files for transcription and translation.

## Supported Formats
- `.mp3` - MPEG Audio Layer 3
- `.wav` - Waveform Audio File Format
- `.m4a` - MPEG-4 Audio
- `.flac` - Free Lossless Audio Codec

## Usage

1. Place your audio file(s) in this folder
2. Run `python main.py` from the project root
3. The application will automatically detect and process audio files in this folder

**Note:** Only the first audio file found will be processed. For multiple files, process them one at a time or modify the code to handle batch processing.
